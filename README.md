# llm-from-scratch

## Running This Code

Run this to activate the Conda environment:

```bash 
$ conda activate llmscratch
```

## Testing 

Tests written using [PyTest](https://docs.pytest.org/en/stable/index.html)

To run tests:

```bash
$ pytest
```
